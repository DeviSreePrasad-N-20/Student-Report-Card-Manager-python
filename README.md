# Student Report Card Manager

A tiny console app that lets you add students, record scores, see averages & grades, delete records, and save everything to JSON.

## How to Run

1. Make sure you have Python 3 installed.
2. Clone this repository or download `Student_Report_Card_Manager.py`.
3. Open a terminal or command prompt in the project directory.
4. Run:


5. Follow the menu in the terminal to add students, record scores, view reports, and save/load your data.

## Features

- **Add students:** Create new student records.
- **Update / add scores:** Record and edit subject scores (0-100).
- **View reports:** Display averages and letter grades for each student.
- **Delete students:** Remove student records.
- **Save/load data:** Automatically saves to and loads from `grades.json`.
- **Easy-to-use menu:** Simple console-based interface.

## Data File

All student data is stored in a JSON file named `grades.json` by default, in the same directory as the script.
